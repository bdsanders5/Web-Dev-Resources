Become an expert in html5, css3, & javascript

This is the start of a collection resources & activities that I hope will expand into a rode map for becoming proficient in all things html5, css3, & javascript. A more rigid schedule that assigns hrs of time to devote on each area shold be forthcoming.

Current Javascript Strategies

  • Start by breaking dificult Javascript concepts into manageable learning topics
  • Study & practice one topic at a time until mastered ie...
    • High level functions like array.reduce(), array.map(), filter, etc
    • Callback Functions
    • .call(), .apply(), .bind()
    • Regex
    • Recursion
    • Binding
    • Closures (http://javascriptissexy.com/understand-javascript-closures-with-ease/
    • Promises, async, await, etc... https://www.youtube.com/watch?v=PoRJizFvM7s
  • Rewrite some python / php in js
  • Study existing code (simplecart, tic-tac-toe, etc)
