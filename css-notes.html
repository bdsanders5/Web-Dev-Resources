Revisit chapter 7 to solidify simple mobile 1st layout

A good breakpoint for media queries is 568px as this catches iphone in landscape mode.

For multiple breakpoints, you can use media queries as such:
/* Normal styles here */ @media (not max-width: 600px) and max-width: 700px) { /* styles for 600 - 700 pixels wide */ } @media (not max-width: 400px) and max-width: 600px) { /* styles for 600 - 700 pixels wide */ } @media (not max-width: 400px) { /* styles for sub 400 pixels wide */ }

Entire css files can be loaded to override styles for smaller devices.

Clearing Floats:

From: https://developer.mozilla.org/en-US/docs/Web/CSS/clear

Note: If an element contains only floated elements, its height collapses to nothing. If you want it to always be able to resize, so that it contains floating elements inside it, you need to self-clear its children. This is called clearfix, and one way to do it is clear a replaced ::after pseudo-element on it.

#container::after { 
          content: "";
          display: block; 
          clear: both;
        }
